# 在线影视库 API Key 配置

刷片夹不会内置任何私人 API Key。需要 Key 的来源请自行申请，然后在 App 的“设置”页填写。

## 已接入来源

- TVMaze：不需要 API Key，首页搜索会直接使用。
- TMDb：填写 `TMDb API Key` 或 `TMDb Bearer Token`。
- OMDb：填写 `OMDb API Key`，可获得 IMDb/OMDb 相关搜索与评分信息。
- Trakt：填写 `Trakt Client ID`。
- Watchmode：填写 `Watchmode API Key`，可搜索流媒体片库。

## 申请入口

- TMDb: https://developer.themoviedb.org/
- OMDb: https://www.omdbapi.com/apikey.aspx
- Trakt: https://trakt.tv/oauth/applications
- Watchmode: https://api.watchmode.com/requestApiKey/
- TVMaze API 文档: https://www.tvmaze.com/api
